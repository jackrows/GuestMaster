brief description:
This is a C console game. The computer has a word and the user must to guest it.
The player dispose limited tries.

Operation:
The game read words from a file and choose random one of them.
The user try to guest this word with limit attempts.
At every word of user there are letters that contains at hidden word, will be appeared.


comments:
this is a small change of hangman game.
Other idea is to implement the Find the Story game. Any proposition ends to 'a' is correct, anything else is wrong.
